Vim
===
My vimrc etc.

Prerequisites
-------------
- Vim >= 7.3, with ruby support
- Development files for the ruby version vim is build with
- ctags

Installation
------------
    \curl -L https://raw.github.com/mkrogh/vim/master/install.sh | bash -s -- [-c] [-d] [-p INSTALL_PATH]

Update
------
    ~/.vim/update.sh
